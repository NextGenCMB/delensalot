Analysis resultsÔÉÅ

delensalot uses an analysis result directory TEMP to store the analysis setting, intermediate and final results. Among those are files related to the noise model, QE and MAP filtering, QE and MAP lensing potentials, and mean-fields.

At the root of TEMP is the the configuration file used in this analysis.

For the noise model delensalot stores,

  • Noise inverted power spectra for both T and P in cinv_t and cinv_p

The QE TEMP dir is TEMP/QE/. For QE lensing reconstruction,

  • lensing potentials in qlms_dd

  • mean-field in qlms_dd

  • inverse variance filtered maps in ivfs

The MAP TEMP dir is TEMP/MAP/. For MAP lensing reconstruction, a realization dependent subdirectory <key>_sim<simidx><V> is made and therefore depends on the users choice of <key> and <V> in the configuration files,

  • a subdirectory hessian with the hessian for each iteration.

  • a subdirectory wflms with the Wiener-filtered lensing potential increments for each iteration and B-lensing templates

  • the QE mean-field mf.npy

  • the realization dependend QE lensing potential phi_plm_it000.npy (with the normalized QE meanfield-subtracted from it)
